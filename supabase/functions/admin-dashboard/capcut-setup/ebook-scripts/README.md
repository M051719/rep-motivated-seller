## Usage

Each chapter has a corresponding script file in the `scripts/` directory:
- Hook (0-3 seconds) - Attention-grabbing opening
- Problem (3-10 seconds) - Define the challenge
- Solution (10-25 seconds) - Present actionable information
- CTA (25-30 seconds) - Clear call to action

## Quick Start

```powershell
# Create video project from a chapter template
.\content-workflow.ps1 new-video "Understanding Pre-Foreclosure" educational

# List all available templates
ls scripts/
```

## Video Series Structure

Each e-book becomes a video series with:
- Consistent branding across all chapters
- Progressive education (builds on previous videos)
- Cross-references to related chapters
- Unified call to action to RepMotivatedSeller.com

## Production Notes

- Target duration: 30-60 seconds per chapter
- Platform: Instagram Reels, YouTube Shorts, TikTok
- Format: Educational, empathetic, solution-focused
- Tone: Professional yet approachable
