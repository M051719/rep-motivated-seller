## â“ Troubleshooting

**"Script won't run"**
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

**"Assets not found"**
```powershell
.\verify-assets.ps1
```
Shows exactly what's missing.

**"Canva backgrounds look wrong"**
- Export as PNG (not JPG)
- Check resolution: 1920x1080
- Re-export if needed

**"No music files found"**
- Place MP3/WAV files in Music, Downloads, or Desktop
- Run organize-all-assets.ps1 again
