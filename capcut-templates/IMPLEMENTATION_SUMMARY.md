- Neutral: #6b7280

**Gradients:** 3 defined gradient combinations for different contexts

### Typography

**Primary:** Inter (6 weights: 300-800)
**Secondary:** Poppins (2 weights: 600, 700)

**Hierarchy:**
- Hero Text: 72-96px (Inter Extrabold)
- Headers: 48-64px (Inter Bold)
- Body: 32-42px (Inter Semibold)
- Small: 24-28px (Inter Medium)

7 logo variations specified:
1. Primary (full color)
2. White (for dark backgrounds)
3. Blue monochrome
4. Icon only
5. Horizontal layout
6. Stacked layout
7. Animated (3-second intro)

**Usage rules:** Clear space, minimum sizes, placement guidelines
