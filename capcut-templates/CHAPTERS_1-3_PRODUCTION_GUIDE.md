## üí° PRO TIPS

### CapCut Shortcuts:
- **Split clip:** Place cursor, press "S"
- **Duplicate:** Select, press "Ctrl+D"
- **Zoom timeline:** Scroll wheel
- **Play/Pause:** Spacebar

### Text Readability:
- Keep font size 48pt+ for 9:16 videos
- Use high contrast (white text on dark, or vice versa)
- Add subtle drop shadow or outline
- Keep text on screen 2+ seconds minimum

### Music Selection:
- Test volume levels on phone speakers
- Ensure voiceover/text can be "heard" over music
- Music should support, not distract
- Always fade in/out (0.5s minimum)

### Mobile Optimization:
- Test on actual phone before publishing
- Phone numbers must be large enough to read
- Avoid clutter (less is more)
- Critical info should be in "safe zone" (center 80%)

## üÜò TROUBLESHOOTING

### "Can't find the .srt file"
- Location: `capcut-templates/subtitles/`
- In CapCut: Text ‚Üí Captions ‚Üí Auto Captions ‚Üí Import SRT

### "Icons look blurry"
- Use PNG files (not JPG)
- Minimum 512√ó512px for icons
- Re-export from Canva at higher resolution

### "Music is too quiet/loud"
- Recommended: 25-30% volume
- Test on phone and computer
- Adjust per scene if needed

### "Text doesn't match timing"
- Check .srt file timestamps
- Manually adjust in CapCut if needed
- Each subtitle should display 2-3 seconds minimum

### "Video exports too large"
- Target: Under 50MB for social media
- Reduce bitrate to 10-12 Mbps
- Keep resolution at 1080 (don't go higher)
