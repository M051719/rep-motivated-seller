## ‚úÖ Quality Standards

### Images (PNG/JPG)

- [ ] Minimum 1920x1080 for backgrounds
- [ ] Transparent PNG for logos/icons
- [ ] sRGB color profile
- [ ] High quality export (no compression artifacts)
- [ ] Properly named files

### Videos (MP4)

- [ ] H.264 codec
- [ ] 1920x1080 minimum resolution
- [ ] 30fps frame rate
- [ ] 15-20 Mbps bitrate
- [ ] Seamless loops (if looping)

### Audio (MP3)

- [ ] Music: 320kbps, 48kHz, stereo
- [ ] SFX: 192kbps, 48kHz
- [ ] Normalized to -3dB peak
- [ ] No clipping or distortion
- [ ] Proper fade in/out

### Fonts (TTF/OTF)

- [ ] Complete character set
- [ ] Proper kerning
- [ ] Licensed for commercial use
- [ ] Tested in CapCut
