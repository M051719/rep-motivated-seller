## Asset Organization Tips

### Folder Structure in CapCut
When importing to CapCut, organize like this:
```
Media/
├── Logos
├── Backgrounds
├── Icons
├── B-Roll
├── Interview Footage
└── Graphics
```

### Naming Convention
Use descriptive names:
- Good: `logo-white-transparent.png`
- Good: `bg-gradient-blue-purple.png`
- Bad: `image1.png`
- Bad: `file-final-v2.png`

### Best Practices
1. **Keep originals:** Always keep master files (PSD, AI) separate
2. **Version control:** Use v1, v2 for iterations
3. **Backup:** Keep copy in cloud storage
4. **Organize by project:** Create folders for each video project
5. **Delete unused:** Clear out assets you won't use

## Asset Creation Guidelines

### Creating New Assets

**Images:**
- Format: PNG (transparent) or JPG (photos)
- Resolution: Minimum 1920x1080 for backgrounds
- Color profile: sRGB
- Export at high quality

**Videos:**
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (Full HD)
- Frame rate: 30fps
- Bitrate: 15-20 Mbps
- No audio unless needed

**Audio:**
- Format: MP3 (music) or WAV (SFX for editing)
- Sample rate: 48kHz
- Bit depth: 24-bit (WAV) or 320kbps (MP3)
- Normalize to -3dB peak

## CapCut Import Guide

### How to Import Assets

1. **On Desktop:**
   - Click "Media" tab
   - Click "Import" button
   - Select files or drag-and-drop
   - Assets appear in Media library

2. **On Mobile:**
   - Tap "+" to create new project
   - Tap "Photos" or "Videos"
   - Select from device library
   - Or use "Import from cloud"

### Organizing in CapCut

1. Create folders for different asset types
2. Use tags/labels for quick finding
3. Star frequently used assets
4. Delete unused media to save space

### Asset Limits
- Desktop: No practical limit
- Mobile: Depends on device storage
- Cloud: 10GB free, more with subscription

## Troubleshooting

### Common Issues

**Problem:** Custom fonts not showing
- **Solution:** Reinstall fonts, restart CapCut, use system fonts

**Problem:** Video lag/stuttering during editing
- **Solution:** Use proxy/draft mode, reduce preview quality

**Problem:** Audio out of sync
- **Solution:** Convert audio to 48kHz before importing

**Problem:** Transparent PNG showing black background
- **Solution:** Ensure PNG has alpha channel, reimport

**Problem:** Large file sizes
- **Solution:** Compress videos before import, use H.264 codec
