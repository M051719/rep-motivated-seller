===========================================
HOW TO CREATE IN CANVA (Free):
===========================================

1. Go to: canva.com
2. Click "Create a design" → "Custom size"
3. Enter: 1920 x 1080 pixels
4. Click "Create new design"

FOR GRADIENTS:
5. Click "Elements" → Search "gradient background"
6. Or click background → "Color" → "Gradient"
7. Click gradient editor
8. Set colors using hex codes above
9. Adjust angle using the rotation handle
10. Download: File → Download → PNG

FOR SOLID COLORS:
5. Click background
6. Click "Color"
7. Click "+" to add custom color
8. Enter hex code (e.g., #2563eb)
9. Download: File → Download → PNG


===========================================
HOW TO CREATE IN PHOTOSHOP:
===========================================

1. File → New
2. Width: 1920px, Height: 1080px
3. Resolution: 72 ppi
4. Color Mode: RGB

FOR GRADIENTS:
5. Select Gradient Tool (G)
6. Click gradient preview in top toolbar
7. Click color stops, enter hex codes
8. Set angle in top toolbar
9. Click and drag to apply
10. File → Export → Export As → PNG

FOR SOLID COLORS:
5. Edit → Fill
6. Select "Color"
7. Enter hex code
8. File → Export → Export As → PNG


===========================================
HOW TO CREATE ONLINE (Easiest):
===========================================

Use: https://cssgradient.io/
1. Select "Linear Gradient"
2. Set angle
3. Add color stops with hex codes
4. Click "Copy CSS"
5. Then use https://www.gradient-animator.com/
   - Paste CSS
   - Export as image (1920x1080)

Or use: https://www.canva.com/ (recommended for beginners)
