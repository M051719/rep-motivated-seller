## üöÄ FASTEST METHOD: Canva + Free Icons

### Step-by-Step (5 minutes per icon):

**1. Set Up Template**
```
1. Go to: canva.com
2. Create custom size: 512 x 512 px
3. Background: Leave transparent
4. Save as template for reuse
```

**2. Find & Customize Icons**
```
1. Click "Elements"
2. Search for icon name (e.g., "house", "document", "phone")
3. Filter by "Graphics" ‚Üí "Simple"
4. Select a clean, modern icon
5. Resize to fill canvas (leave 50px padding)
6. Change color to brand blue (#2563eb) or white (#ffffff)
7. Download: PNG with transparent background
```

**3. Name and Save**
```
Save as: icon-[name].png
Example: icon-foreclosure.png
Save to: /capcut-templates/assets/icons/
```

## üé® ALTERNATIVE: Create Custom Icons (Advanced)

### Using Figma (Free):

**1. Set Up**
```
1. Go to: figma.com (free account)
2. Create new file
3. Create frame: 512 x 512 px
4. Name: icon-[name]
```

**2. Create Icon**
```
Use tools:
- Rectangle (R) for squares
- Ellipse (O) for circles
- Pen tool (P) for custom shapes
- Text (T) for letter-based icons

Keep it simple:
- Solid fills or strokes
- 4-8px stroke width
- Rounded corners (8-12px radius)
- Center aligned
```

**3. Export**
```
1. Select frame
2. Export settings:
   - Format: PNG
   - Size: 1x (512px)
3. Export
4. Save to assets/icons/
```

## üìê ICON STYLE GUIDE

### Visual Consistency:

**All icons should:**
- Use same line weight (4-6px)
- Have similar level of detail (simple to moderate)
- Follow same style (outline OR solid, not mixed)
- Use same corner radius
- Be optically centered
- Have consistent padding (40-50px from edges)

**Example Style Options:**

**Option A: Outline Style**
```
- Line weight: 5px
- No fill, just stroke
- Rounded corners
- Modern, clean look
```

**Option B: Solid Style**
```
- Filled shapes
- No outlines
- Bold, impactful
- Good for small sizes
```

**Choose ONE style and stick to it across all icons.**

## üÜò TROUBLESHOOTING

**Icons look pixelated:**
- Ensure you're creating at 512x512, not smaller
- Don't upscale small icons
- Use vector icons (SVG) when possible

**Can't find right icon:**
- Try different search terms
- Browse icon category pages
- Check multiple icon sites
- Use AI description: "simple outline house icon"

**Colors don't match:**
- Use exact hex codes from brand guidelines
- Check in different lighting
- Preview on actual dark/light backgrounds
