## üé¨ BONUS: Create Your First Test Video (30 min)

**Now that you have assets, make a quick 30-second video:**

### Step-by-Step:

**1. Create New Project (2 min)**
- Open CapCut
- Click "New Project"
- Name it: "Test Video 01"

**2. Import Assets (3 min)**
- Click "Import"
- Select:
  - gradient-blue-indigo.png
  - RepMotivatedSeller_Logo_White.png
  - Your music track
  - Any video footage you have (or use stock from CapCut)

**3. Build Structure (25 min)**

**Intro (0-3s):**
```
- Drag gradient background to timeline
- Drag logo on top (overlay track)
- Center logo
- Add fade-in animation:
  - Select logo ‚Üí Animation ‚Üí In ‚Üí "Fade"
- Duration: 3 seconds
```

**Main Content (3-25s):**
```
- Add another background or video footage
- Click "Text" ‚Üí Add title
- Type: "Facing Foreclosure?"
- Font: Inter Bold, 72px, White
- Position: Center
- Animation: Fade + Scale
- Duration: 5 seconds

- Add second text:
  Type: "We Can Help"
  Font: Inter Bold, 56px, White
  Position: Below first text
  Duration: 5 seconds
```

**CTA (25-30s):**
```
- Add gradient background again
- Add text: "Get Free Consultation"
- Font: Inter Bold, 64px
- Add text below: "www.RepMotivatedSeller.com"
- Font: Inter SemiBold, 36px
- Duration: 5 seconds
```

**4. Add Music (5 min)**
```
- Drag music track to audio track
- Trim to 30 seconds
- Set volume: -18dB
- Add fade in (1s) and fade out (2s)
```

**5. Export (2 min)**
```
- Click "Export"
- Settings:
  - Resolution: 1080p
  - Frame rate: 30fps
  - Format: MP4
  - Quality: High
- Click "Export"
- Save as: "test-video-01.mp4"
```

‚úÖ **DONE! You made a video!**

## üÜò TROUBLESHOOTING

### "Canva won't let me download PNG"
- **Free account:** Limited to 10 downloads/day
- **Solution:** Create free account, or upgrade to Pro ($13/month)
- **Alternative:** Use Photopea.com (free Photoshop alternative)

### "Fonts don't appear in CapCut"
- **Solution:** Restart CapCut after installing fonts
- **Check:** Fonts installed on system (not just downloaded)
- **Alternative:** Use system fonts as backup

### "Don't have video footage"
- **Solution:** Use CapCut's stock library (free)
- **Or:** Pexels.com, Pixabay.com (free stock video)
- **Or:** Use static images with Ken Burns effect (zoom)

### "Music is copyrighted"
- **Solution:** Only use YouTube Audio Library or licensed tracks
- **Check:** License allows commercial use
- **Safe option:** Epidemic Sound ($15/month unlimited)
