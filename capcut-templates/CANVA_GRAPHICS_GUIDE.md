## üìä GRAPHIC 1: TIMELINE GRAPHIC (Chapter 1)

### Purpose:
Show foreclosure stages at 30/90/120+ days

### Canva Steps:

**1. Create New Design:**
- Go to Canva.com ‚Üí Create a design
- Click "Custom size"
- Enter: 1080 width x 1920 height
- Click "Create new design"

**2. Add Background:**
- Click "Elements" ‚Üí Search "gradient"
- Choose: Blue to Purple gradient (or create custom)
- Custom gradient:
  - Top color: #3b82f6 (blue)
  - Bottom color: #7c3aed (purple)
  - Direction: Vertical
- Drag to fill entire canvas

**3. Add Title (Top):**
- Click "Text" ‚Üí Add heading
- Type: "Foreclosure Timeline"
- Font: Poppins Bold
- Size: 64pt
- Color: #ffffff (white)
- Position: Top center, 100px from top
- Add text shadow: Offset 2px, Blur 4px, Transparency 30%

**4. Create Timeline Line:**
- Click "Elements" ‚Üí Search "line"
- Choose straight horizontal line
- Color: #ffffff (white)
- Width: 800px
- Thickness: 8px
- Position: Horizontal center, 500px from top

**5. Add Timeline Markers (3 total):**

**Marker 1 - 30 Days:**
- Click "Elements" ‚Üí Search "circle"
- Choose solid circle
- Color: #10b981 (green)
- Size: 80px diameter
- Position: Left side of line (200px from left edge)
- Add text above circle:
  - "30 DAYS"
  - Font: Poppins SemiBold, 42pt
  - Color: #ffffff
  - Position: Centered above circle, 40px spacing
- Add text below circle:
  - "First Missed Payment"
  - Font: Inter Medium, 32pt
  - Color: #ffffff
  - Position: Centered below circle, 40px spacing

**Marker 2 - 90 Days:**
- Duplicate Marker 1
- Move to center of line (540px from left)
- Change circle color to: #f59e0b (orange)
- Change text:
  - Top: "90 DAYS"
  - Bottom: "Lender Can Start"

**Marker 3 - 120+ Days:**
- Duplicate Marker 1
- Move to right side of line (880px from left)
- Change circle color to: #ef4444 (red)
- Change text:
  - Top: "120+ DAYS"
  - Bottom: "Auction Possible"

**6. Add Bottom Message:**
- Click "Text" ‚Üí Add subheading
- Type: "You Have Options at Every Stage"
- Font: Poppins SemiBold, 48pt
- Color: #ffffff
- Position: Bottom center, 200px from bottom
- Add slight glow effect

**7. Export:**
- Click "Share" ‚Üí "Download"
- File type: PNG
- Quality: High
- Filename: `chapter-01-timeline-graphic.png`
- Download

**Save Location:**
`Content-Creation/Assets/graphics/chapter-01-timeline-graphic.png`

## üí° PRO TIPS

### 1. Use Templates:
- Search Canva for "Instagram Story" or "TikTok" templates
- Filter by "Reels" or "Stories"
- Modify colors/text to match specs

### 2. Brand Kit (Canva Pro):
- Save color palette as brand colors
- Upload fonts if not available
- Create reusable templates

### 3. Batch Creation:
- Create one design
- Duplicate for variations
- Modify text/colors
- Saves time on setup

### 4. Organization:
- Create folders in Canva:
  - "Chapter 1 - Foreclosure"
  - "Chapter 2 - Emotions"
  - "Chapter 3 - Communities"
- Name files consistently

### 5. Quality Check:
- Zoom to 100% before exporting
- Check text legibility on mobile
- Verify colors match brand
- Test on dark/light backgrounds
