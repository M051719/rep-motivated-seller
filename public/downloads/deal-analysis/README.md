# Deals and Dollars Resources

This directory contains real estate investment analysis tools and templates from the Al Aiello LLC Master Machine Asset Protection System.

## Contents

### Excel Deal Analyzers
- **Amortization Table.xls** - Loan amortization calculator
- **Deal Analysis Software.xlsx** - General deal analysis tool
- **Deal+Analyzer+for+Flips.xlsx** - Specialized analyzer for flip properties
- **Deal+Analyzer+for+Rentals+BASIC.xlsx** - Basic rental property analyzer
- **Deal+Analyzer+for+Rentals+FULL.xlsx** - Full-featured rental property analyzer
- **Flip-Or-Rent-Calculator-from-Rehab-Valuator-1.xlsx** - Decision tool for flip vs rent strategy
- **Performance Dashboard-2.xls** - Property performance tracking dashboard
- **Repair+Estimator+Worksheet.xlsx** - Repair cost estimation tool
- **underwriting Deal Analysis Software.xlsx** - Interactive underwriting analysis

### Document Templates
- **Updated 3 Option LOI with Owner Finance Language.docx** - Letter of Intent template with owner financing options
- **CAP RATES.docx** - Cap rate reference document
- **Quarterly_Cap_Rate_Survey_D_D_Club_edited_by_Lance_updated2.xlsx** - Market cap rate data

## Usage

These tools are designed for analyzing real estate investment opportunities including:
- Property flips
- Rental properties
- Owner financing deals
- Deal underwriting
- Performance tracking
- Repair estimation
